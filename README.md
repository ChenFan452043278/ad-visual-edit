# ad-visual-edit

## Installation

### With npm

```bash
npm i ad-visual-edit
```

### Props
#### list
Type: `Array`<br>
Required: `true`<br>
Default: `null`

广告列表的数据

#### uploadUrl
Type: `String`<br>
Required: `true`<br>
Default: `null`

对应项目上传图片的地址

#### postModule
Type: `Function`<br>
Required: `true`<br>
Default: `null`

获取编辑完成的广告列表数据
